# Lazara Website Mockup

A simple React + Vite + Tailwind homepage mockup for Lazara.

## How to run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in your terminal.

## How to deploy

The easiest route is Vercel:

1. Upload this folder to GitHub.
2. Connect the GitHub repo to Vercel.
3. Deploy using the default Vite settings.

## Notes

This is a front-end mockup only. Shop, quiz, account and journal links are placeholders.
